# Formulário de Matrículas - Estrelas do Amanhã

Este é um projeto de formulário de matrículas para a escola de educação infantil **Estrelas do Amanhã**. O objetivo do projeto é fornecer uma interface amigável e acessível para que os pais ou responsáveis possam matricular seus filhos na escola.

## 📋 Funcionalidades

- Formulário dividido em seções:
  - Informações da criança
  - Endereço residencial
  - Informações do responsável
  - Opções de matrícula
- Upload de arquivos (Certidão de Nascimento)
- Validação de campos obrigatórios (exemplo: e-mail)
- Seleção de esportes e turnos
- Aceite dos Termos e Condições
- Botões para salvar respostas ou finalizar a matrícula

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura do formulário e organização semântica do conteúdo.
- **CSS3**: Estilização do formulário e layout responsivo.
  - Arquivos CSS organizados em:
    - `styles/index.css`: Estilos principais.
    - `styles/forms.css`: Estilos específicos para formulários.
    - `styles/fields/`: Estilos para botões, checkboxes, inputs, etc.
- **Google Fonts**: Fontes personalizadas para melhorar a aparência do texto.
  - Fontes utilizadas: Alice, Archivo, Asap, Inconsolata, Maven Pro, Open Sans, Poppins.
